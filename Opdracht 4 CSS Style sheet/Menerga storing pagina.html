Welkom op de Menerga pagina voor storingen aan uw machine

Op deze pagina kan een menu worden doorlopen. Dit doen wij om de storing zo gericht mogelijk door te kunnen geven aan onze storing monteur.

Selecteer hier uw type machine.

Deze word als volgd aangeduid, 29/15/01.
De 1e cijfers staan voor het type machnine.
De 2e cijfers staan voor het aantal cubike meters lucht.
De 3e cijfers staan voor bijzonder aanduiding.
01 binnen opstelling.
